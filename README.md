### 🛒 E-commerce con React

<a href="https://e-commerce-xi-tawny-82.vercel.app">
  <img width="45%" alt="E-commerce" src="https://github.com/RoberDev96/E-Commerce/blob/main/public/Captura%20de%20pantalla%20de%202026-05-02%2011-35-40.png?raw=true" />
</a>

**Tienda online funcional** - Carrito de compras, autenticación y productos desde API real.

**✨ Características:**
- Autenticación de usuarios (Context API + localStorage)
- Rutas protegidas (PrivateRoute)
- Listado de productos desde FakeStore API
- Filtrado por categorías
- Carrito de compras (Context + localStorage)
- Formularios con React Hook Form + Zod
- Fetching con TanStack Query (caché, loading, errores)

**🛠️ Tecnologías:**
- React 18 | TypeScript | React Router | TanStack Query | TailwindCSS | FakeStore API

**🔗 Enlaces:**
- [Demo en vivo](https://e-commerce-xi-tawny-82.vercel.app)
